6.18.1  Record 6C Cell and Latron Lot Numbers in Integrated Survey 
Information System  
The Integrated Survey Information System ( ISIS) maintains the capability to download all 6C cell 
and Latron control data. The data are used to monitor quality control results. Upload or enter data 
for each lot number each time a new lot number is put into use. The ISIS uses the same 6C cell 
control package insert and Latron package insert as the DxH  800.
